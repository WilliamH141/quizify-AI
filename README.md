# Quizify AI 

AI-powered quiz builder built with Next.js, Prisma, and Supabase.  
Currently a work in progress - setting up database, authentication, and OpenAI integration.

## Setup
1. Install dependencies:  
   `npm install`
2. Add environment variables in `.env`:  
   `DATABASE_URL=your_supabase_url`  
   `DIRECT_URL=your_supabase_direct_url`
3. Push Prisma schema:  
   `npx prisma db push`
4. Run the dev server:  
   `npm run dev`
